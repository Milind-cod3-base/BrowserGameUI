## Build 🛠️

1. Clone:  
   ```bash
   git clone https://github.com/Milind-cod3-base/BrowserGameUI.git
   cd BrowserGameUI
   ```
   
2. Install:
	```bash

	npm install
	```

3. Run:
	```bash
		npm run start
	```
## Features 🌟
	Navigation Bar: Home, Map, Stats.
	Stats Card: Quests, Achievements, Skills.
	Circular Menu: Start, Load, Settings, Credits.
